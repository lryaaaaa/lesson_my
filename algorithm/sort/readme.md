- 冒泡排序  Bubble Sort
  1. 排序
    很多种，
  2. 交换
  3. 相邻
  4. 有序区域 j < length-i-1  ,第一波优化
  5. 稳定排序  算法的可靠性
  6. 识别有序区域的边界？
      最后交换的位置？

    交换排序（位置，从小到大，从大到小）
- 求最值，不能传整个数组，只传字符串
    - Math.min()求最小
    - Math.max()求最大

- 如何写一个高质量的冒泡排序？
 1. 当在一次冒泡的内层循环中，一次交换都没有发生，
    通过本轮是否有交换？ isSorted 布尔值
      如果没有，那么待排序就已有顺序，可以退出 
 2. 基于isSorted再优化
    如果后半段有序，还可以再优化
